Local Maximum in Puzzles

Game is about problem-solving. Problem-solving forms puzzles. I noticed that some puzzles have only one maximum, while others have multiple maximums. Here by "maximum" I mean a status of the puzzle that no more action could make it closer to the final goal. The status where the final goal is reached is definitely a global maximum with this definition, while other maximums are so-called "local maximum". For example, in a Sokuban level like the picture following, if the player push the red box first to the bottom, he will never acheive his final goal, so he enters a local maximum of the puzzle.

I think this classification can be useful when design puzzles. It abstructs the puzzles into their math nature so we could use math tools on them. It also can be used to predict players' behaviors. I will use some instances to show how to abstruct puzzles with the len of maximum, and how palyers make different strateties related to their maximums.
