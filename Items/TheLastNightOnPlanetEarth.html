The Last Night on Planet Earth

  • Platform: PC
  • Tool: Unity 3D
  • Role: UI Programmer, UI Designer
  • Employer: Dejobaan Games

The Last Night on Planet Earth is an artistic visual novel by Dejobaan Games. It tells an interesting story of an alien on an earth about to explode. It is full of dark humor and unique views that come from the alien aspect. I started the Programming Intern in the team in May 2018, when the game had an early version written in Python. The goal is to make a fully completed and polished version in Unity. Specifically, my job is to design and implement the UI system in Unity, because the Python version simply displays pictures with no UI effects.

This intern brings me new and exciting challenges. To design and implement UI for such a beautiful artistic design, I am researching in UI Animation, Shader and other useful skills. It is so fun and enjoyable to play with those beautiful elements.
